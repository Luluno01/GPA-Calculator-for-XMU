# 计算方法
## 不参与计算的课程
* 成绩为`合格`或`不合格`的；
* `免修`的；
* `全校性选修课`；
* 不知道怎么算的（没成绩的？）。

## 参与计算的单科计算标准
| 分数区间 | 等级 | 绩点 |
|---------|------|------|
| [90, +∞)|   A  |  4.0 |
| [85, 89]|   A- |  3.7 |
| [81, 84]|   B+ |  3.3 |
| [78, 80]|   B  |  3.0 |
| [75, 88]|   B- |  2.7 |
| [72, 74]|   C+ |  2.3 |
| [68, 71]|   C  |  2.0 |
| [64, 67]|   C- |  1.7 |
| [60, 63]|   D  |  1.0 |
| [60, -∞)|   F  |   0  |
|  `NaN`  | 未知 |   0  |

## 总绩点计算方法
```
学期总绩点 = (∑(i = 1, n)(单科绩点 * 单科学分)) / ∑(i = 1, n)(单科学分)
```
其中，n为参与绩点计算的课程数。本公式适用于学期绩点和总绩点的计算。
